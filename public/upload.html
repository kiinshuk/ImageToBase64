Upload a File


Description

This tool allows you to upload a file and convert it to Base64 format, or convert a Base64 string back to its original file format.

How to Use

  • To convert a file to Base64: Upload the file using the form above and view the Base64 string.
  • To convert Base64 to a file: Click the button to go to the conversion page, paste the Base64 string, and download the converted file.
